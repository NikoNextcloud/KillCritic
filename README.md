# KILLCRITIC

Свежо, mobile-first уеб приложение за преминаване през желание за алкохол без вина, броене на „провали“ или назидание.

## Какво работи

- четири ясни основни таба: **Брояч**, **Днес**, **Календар**, **Още**
- жив брояч с цветно запълване по години, месеци, дни, часове, минути и секунди
- старт с бутон **Трезвеност**, спиране или нулиране със запазена история на старите постижения
- 6 дневни предизвикателства от голям набор с рядко повторение и прогрес за деня
- интерактивен месечен календар с текуща серия, рекорд, общ брой и статистика за последните 30 дни
- всички помощни инструменти, AI, GPS, победи и профил са отделени в **Още**
- 5-минутен кризисен режим с дишане, интензитет и визуална вълна
- лично гласово послание чрез микрофона
- дневник на настроенията без писане
- библиотека на причините и локални прозрения
- симулатор „След 6 часа“
- ежедневни мисии и фокус върху спечеленото
- човек за контакт, експорт и изчистване на данните
- данните се пазят само в браузъра чрез `localStorage`
- всеки нов профил започва с 0 победи и празна история
- рискови GPS места с избираем предупредителен радиус
- ръчно въвеждане на пари, загубен сън, енергия, тренировки, тревожност и продуктивност
- радар за повтарящи се оправдания, планирано и реално количество
- по-дълбок AI анализ през защитен serverless endpoint

## Публикуване

Качи съдържанието на тази папка в GitHub repository. Обикновените функции работят и в GitHub Pages, но GitHub Pages няма защитен backend и AI бутоните ще използват локалния анализ.

За пълната AI версия:

1. Импортирай GitHub repository-то във Vercel.
2. Създай безплатен API ключ в [Groq Console](https://console.groq.com/keys).
3. Във Vercel, **Settings → Environment Variables**, добави `GROQ_API_KEY`.
4. По желание добави `GROQ_MODEL`; стандартно се използва `openai/gpt-oss-120b`.
5. Натисни **Deploy** отново. Ключът остава на сървъра и никога не се записва в браузъра.

AI заявките изпращат само записите, нужни за анализа. Точните GPS координати не се изпращат към AI.

## Локален преглед

При наличен Node.js стартирай `node dev-server.js` и отвори `http://127.0.0.1:8788`. Не отваряй само `index.html`, защото микрофонът, GPS и инсталирането като приложение изискват защитен адрес или localhost.

Микрофонът и GPS работят на `https://` или `localhost`. GPS предупрежденията на тази уеб версия работят, докато приложението е отворено. За истински фонови предупреждения при затворено приложение е необходима нативна Android/iOS версия.

## Важно

Това е помощен инструмент, не медицинска услуга или заместител на лечение. При непосредствен риск се обадете на 112. Рязкото спиране на алкохола може да бъде опасно при физическа зависимост и изисква медицинска преценка.

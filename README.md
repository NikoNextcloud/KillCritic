# KILLCRITIC

Личен помощник за трудните моменти при отказ от алкохол — без вина и без назидание.

Този проект е React + TypeScript + Tailwind v4 + Vite миграция на оригиналния
статичен KillCritic (HTML/CSS/JS), пренесен във визуалния стил (тъмна,
„стъклена“ премиум тема с неонови акценти) на по-ранния проект AlcoholKill,
като цялата функционална логика на KillCritic е запазена непроменена.

## Функционалност

- **Брояч на трезвеност** — живо отброяване в години/месеца/дни/часове/минути/секунди,
  с история на предишни постижения.
- **Ежедневни предизвикателства** — 6 произволно (но детерминистично за деня) избрани
  малки действия от пул от 50 варианта.
- **Календар на трезвеността** — месечен изглед с отбелязване на трезви/пияни дни,
  серии и обща статистика.
- **„Имам желание“ / Вълната** — 5-минутен кризисен режим с дихателно упражнение,
  интензитет на желанието, лично аудио послание и бърз достъп до контакт/112.
- **Настроение, причина, мисия на деня** — бързи еднодокосвени отбелязвания.
- **Истинската цена на навика** — ръчно въвеждане на сън, енергия, тревожност,
  тренировки, продуктивност и пари за всеки случай.
- **AI радар за самозаблуда** — сравнява текущо оправдание с лична история;
  ползва Groq AI бекенд (`/api/ai`), с локален fallback анализ.
- **Рискови места** — GPS наблюдение и предупреждение при доближаване до запазени
  локации.
- **Профил и поверителност** — всички данни се пазят локално (localStorage);
  експорт на JSON архив; нов профил/нулиране при нужда.

## Технологичен стек

- React 19 + TypeScript
- Tailwind CSS v4 (чрез `@tailwindcss/vite`)
- Vite 6
- lucide-react за икони
- Бекенд: Vercel serverless функция в `api/ai.js`, ползваща Groq API
  (изисква `GROQ_API_KEY` environment variable)

## Стартиране

```bash
npm install
npm run dev
```

## Production build

```bash

```
